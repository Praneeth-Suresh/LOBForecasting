# Mid-Price Forecasting

The goal of this project is to apply interpretability techniques to mid-price forcasting, as a demonstration of the use that different techniques have.

## Getting Started

This project is based on the Benchmark dataset by Ntakaris et al. which is available through Kaggle.

The dataset can be downloaded from [kaggle here](https://www.kaggle.com/datasets/praanj/limit-orderbook-data?resource=download "dataset"). Once downloaded, create a Data folder in the root directory, and populate it with the _text_ files that make up the repository.

Here is the research paper with full details about this dataset: Ntakaris, Adamantios, et al. "Benchmark dataset for mid‐price forecasting of limit order book data with machine learning methods." _Journal of Forecasting_ 37.8 (2018): 852-866.
